# AI-Driven Adaptive Onboarding System

This project contains the full-stack structure for an AI-driven adaptive onboarding system.

## Project Structure

- `frontend/`: React application containing components, pages, and services.
- `backend/`: Node.js (Express) server handling resume parsing, skill extraction (LLM), and gap analysis.

## Features

- Resume upload
- Skill extraction using LLM
- Gap analysis
- Personalized learning roadmap generation
